# Somaliland Disease Surveillance Dashboard

## Overview

This dashboard visualizes disease surveillance data from HIS Excel exports.

## Data pipeline

raw Excel → Python cleaning/aggregation → data/dashboard_data.json → frontend rendering

## Run locally

python3 -m http.server 8000

Open:

http://localhost:8000/

## Rebuild dashboard data

python scripts/build_dashboard_data.py \
  --input "raw_data/All Hospitals Data for last Three month.xlsx" \
  --definitions "data/disease_definitions.json" \
  --hospital-regions "data/hospital_regions.json" \
  --output "data/dashboard_data.json" \
  --expected-hospitals 12
